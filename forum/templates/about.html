Главная О нас Статус API

Это веб-форум, созданный на Go с использованием SQLite. Форум поддерживает:

  • Регистрацию и авторизацию пользователей
  • Создание постов с категориями
  • Комментирование постов
  • Лайки и дизлайки для постов и комментариев
  • Фильтрацию постов по категориям, авторам и лайкам

Технологии: Go, SQLite, HTML, CSS, JavaScript
